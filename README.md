# Sistema de Controle de Estoque

## Descrição

Projeto desenvolvido em Python para gerenciamento de estoque pelo terminal. O sistema permite cadastrar produtos, remover produtos, alterar quantidades e consultar o estoque.

Os dados são armazenados apenas em memória durante a execução do programa.

---

## Funcionalidades

* Adicionar produto
* Remover produto
* Alterar quantidade
* Consultar estoque
* Validação de entrada
* Tratamento de erros
* Testes básicos

---

## Tecnologias Utilizadas

* Python 3

---

## Estrutura do Projeto

```text
sistema-estoque/

├── main.py
├── produto.py
├── estoque.py
├── tests.py
└── README.md
```

* **main.py**: menu principal e interação com o usuário.
* **produto.py**: classe Produto.
* **estoque.py**: gerenciamento dos produtos.
* **tests.py**: testes básicos do sistema.
